 Data Professional Survey Breakdown — Power BI Dashboard

A one-page Power BI dashboard analyzing survey responses from 630 data professionals, exploring salary, career happiness, programming language preferences, and how difficult it is to break into the data field.

This is my first end-to-end Power BI project, covering data cleaning, DAX measures, and dashboard design.

 🗂️ About the Dataset

The dataset contains **630 survey responses** and **28 columns**, covering:
- Current job title, industry, and salary range
- Career switch history (did the respondent switch into data?)
- Favorite programming language
- Happiness ratings (1–10) across salary, work/life balance, coworkers, management, upward mobility, and learning opportunities
- Difficulty rating for breaking into the data field
- Demographics: age, gender, country, education level, ethnicity

 🔍 What the Dashboard Shows

**Key Metrics (Cards)**
- Total count of survey takers
- Average age of survey takers

**Visuals**
- **Average Salary by Job Title** — bar chart comparing pay across roles (Data Analyst, Data Engineer, Data Scientist, etc.)
- **Favorite Programming Language** — column chart of the most-used languages among respondents
- **Country of Survey Takers** — treemap showing the geographic spread of respondents
- **Happiness with Work/Life Balance** — gauge visual
- **Happiness with Salary** — gauge visual
- **Difficulty to Break Into Data** — donut chart showing how hard respondents found it to enter the field

 🛠️ Tools & Skills Used

- **Power BI Desktop** — data modeling, DAX measures, dashboard/report design
- **Power Query** — data cleaning and shaping
- **Excel** — source data review

💡 Key Takeaways

- Salary varies significantly by job title, with more senior/specialized roles commanding higher average pay.
- Python is the clear favorite among the listed programming languages, though R and SQL also have strong representation.
- Survey respondents are geographically concentrated in a few countries, with the U.S. representing a large share.
- Perceived difficulty of breaking into data is mixed — a meaningful portion of respondents found it "Difficult" or "Very Difficult," highlighting the competitiveness of the field.
